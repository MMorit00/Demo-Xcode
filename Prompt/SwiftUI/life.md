我将通过以下思考链分析每个视图：

1. 容器实例分析 📦

```thinking

```

2. 父视图生命周期 👨
━━━━━━━━━━━━━━━━━━
```thinking
- 父视图是如何创建的？

- 父视图在什么条件下销毁？
- ⭐️ 父视图是否持有特殊的持久化状态？
```

3. 子视图生命周期 👶
━━━━━━━━━━━━━━━━━━
```thinking
- 子视图是按需创建还是预创建？

- 子视图在什么条件下被销毁？
- ⭐️ 子视图销毁时是否有状态需要保持？
```
